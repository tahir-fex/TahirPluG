Termux

Download Termux from the link above. Open it and run:

pkg update && pkg upgrade -y

Whenever it prompts y/n, type y and press enter. Wait until it finishes.

Give storage permission:

termux-setup-storage

Install Proot-Distro:

pkg install proot-distro -y

After installation, install Ubuntu using:

pd i ubuntu

Once installation is complete, start Ubuntu:

pd sh ubuntu

Now you're inside Ubuntu. Next, you can proceed with the VS Code Server tutorial below.

How to Run VS Code Server on Android

This tutorial guides you through installing and running code-server on Android using Termux.

Download VS Code Server

wget https://github.com/coder/code-server/releases/download/v4.103.1/code-server-4.103.1-linux-arm64.tar.gz

Extract the Archive

tar -xvf ./code-server-4.103.1-linux-arm64.tar.gz

Navigate to the Code-Server Binary

cd code-server-4.103.1-linux-arm64
cd bin

Set Your Password

export PASSWORD="yourpassword"

Replace yourpassword with a strong password of your choice.

Run VS Code Server

./code-server
